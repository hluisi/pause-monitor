# idx-home

## Startup Sequence
1. Confirm active Serena project via `get_current_config`.
2. Set `session_home_project` from startup active project.
3. If no active project exists, ask user to choose/create and activate one.
4. Read `idx-home`.
5. Load only relevant memories for the task.
